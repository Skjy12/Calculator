# Calculator
A simple web-based calculator with basic arithmetic operations.
## Features
- Addition (+)
- Subtraction (-)
- Multiplication (x)
- Division (/)
- Clear (CL) and Backspace (DEL) buttons
## Demo
Try it live at [Demo](https://your-demo-link.com).

## Usage
1. Clone the repository.
2. Open `indes.html` in a web browser.
